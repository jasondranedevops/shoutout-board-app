AUTHENTICATION METHODS
======================

1. JWT Bearer Token
   - 7-day expiry
   - Generated via /auth/register or /auth/login
   - Included in Authorization: Bearer <token>

2. API Keys
   - Generated via dashboard (admin only)
   - Bcrypt hashed storage
   - Scoped permissions
   - Revocable
   - Included in X-API-Key header

WEBHOOK EVENTS
==============

1. board.created
   - Triggered when new board created
   - Payload: boardId, title, recipientName, createdAt

2. board.sent
   - Triggered when board sent to recipient
   - Payload: boardId, title, recipientName, sentAt

3. post.created
   - Triggered when new post added
   - Payload: boardId, postId, authorName, isAnonymous, createdAt

4. post.deleted
   - Triggered when post removed
   - Payload: boardId, postId

All webhooks signed with HMAC-SHA256 signature in X-Shoutboard-Signature header.
Retry strategy: exponential backoff up to 3 attempts.
